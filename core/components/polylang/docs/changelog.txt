Polylang 1.1.2
====================================
- Changed:  Chunk tpl.PolylangLinks (added class polylang-toggle)
- Updated:  Snippet PolylangLinks (added parameter "trigger")

Polylang 1.1.1
====================================
- Added: Event OnTogglePolylangLanguage

Polylang 1.1.0
====================================
- Added: Support for PROMT translator
- Added: Support for component msMultiCurrency
- Added: Option "polylang_set_currency_for_language" (allows you to set the default currency for msMultiCurrency based on the language specified).
- Added: Option "polylang_post_processing_translation" (delete extra spaces from modx tags after translation).
- Added: Option "polylang_disallow_translation_completed_field" (disallow translation of a completed field).
- Added: Snippet "PolylangSitemap"
